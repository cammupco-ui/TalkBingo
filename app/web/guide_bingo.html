TalkBingo 게임 규칙

TalkBingo는 대화를 통해 빙고 칸을 채우고 줄을 완성하는 게임입니다. 단순한 빙고가 아닌, 상대방과의 "공감(Approve)"이 핵심입니다.

1. 빙고 셀(칸) 종류

T Type 진실게임 (Truth)

  • 주어진 질문에 솔직한 답변(텍스트/음성)을 입력하세요.
  • 상대방이 답변을 듣고 "공감(Approve)"해야 칸을 획득합니다.
  • 보상: 해당 칸 점유 + EP +1

B Type 밸런스 게임 (Balance)

  • 두 가지 선택지 중 하나를 고르고 이유를 설명하세요.
  • 상대방이 이유에 납득하여 "공감(Approve)"해야 칸을 획득합니다.
  • 보상: 해당 칸 점유 + EP +1

M Type 패자부활전 (Mini Game)

  • 퀴즈 실패 시 칸이 잠김(Lock) 상태가 됩니다.
  • 누구든 잠긴 칸을 누르면 미니게임이 발동됩니다.
  • 미니게임 승자가 해당 칸을 즉시 차지합니다!
2. 승리 조건
  • 빙고 1줄을 먼저 완성하고 [게임 종료]를 선언하면 승리!
  • 이어하기 (Continue)를 선택하면 더 높은 보상에 도전할 수 있습니다.
  • 무승부 (Draw): 동시에 빙고를 완성하거나 칸이 꽉 찰 때까지 승부가 안 나면 보상을 나눕니다.
