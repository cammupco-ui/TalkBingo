# TalkBingo UI 규칙 (UI Rules)

이 문서는 TalkBingo 앱의 주요 UI 동작 및 규칙을 정의합니다.

## 게임 화면 (Game Screen)

### 메세지 티커 (Message Ticker)

메세지 창은 화면 상단(AppBar 아래)에 위치하며, 채팅 메세지를 간략하게 보여주는 역할을 합니다.

#### 1. 높이 및 크기 (Dimensions)
메세지 창의 높이는 화면 전체 높이에 대한 비율(Flex)로 정의됩니다.

*   **기본 상태 (Collapsed):**
    *   **높이:** 전체 화면의 **6%** (Flex 6)
    *   **설명:** 한 줄의 메세지만 표시됩니다.
*   **확장 상태 (Expanded):**
    *   **높이:** 전체 화면의 **15%** (Flex 15)
    *   **설명:** 메세지 전체를 볼 수 있으며, 내용이 길 경우 스크롤이 가능합니다.

#### 2. 확장/축소 아이콘 (Expansion Icon)
우측의 화살표 아이콘을 통해 창을 확장하거나 축소할 수 있습니다.

*   **기본 상태 (접혀있음):** ▲ 위쪽 화살표 (`Icons.keyboard_arrow_up`)
*   **확장 상태 (열려있음):** ▼ 아래쪽 화살표 (`Icons.keyboard_arrow_down`)
    *   *참고: 사용자의 요청에 의해 일반적인 방향과 반대로 설정되었습니다.*

#### 3. 동작 규칙 (Behavior)
*   **자동 축소:** 새로운 메세지를 보내거나 받으면, 메세지 창은 자동으로 **기본 상태(6%)**로 축소됩니다.
*   **스크롤:** 확장 상태에서는 텍스트가 창의 높이를 초과할 경우 스크롤하여 전체 내용을 볼 수 있습니다.

### 채팅 목록 (Chat List)

채팅 리스트에는 일반 채팅 메세지뿐만 아니라, **빙고 보드의 질문과 유저의 선택/응답**도 함께 표시됩니다.

#### 1. 정렬 규칙 (Alignment)
*   **다른 기기로부터 들어오는 메세지 (Incoming):** 왼쪽 정렬 (Left Align)
*   **내가 쓴 채팅글 (Outgoing):** 오른쪽 정렬 (Right Align)
*   **빙고 질문/선택/응답 (Bingo Events):** 중앙 정렬 (Center Align)

#### 2. 빙고 이벤트 스타일 (Bingo Event Style)
빙고 게임 진행 중 발생하는 이벤트(질문, 선택 결과 등)는 턴(Turn)의 주체에 따라 다른 스타일을 적용합니다.

*   **Host (MP)의 턴:**
    *   **배경 (Background):** `--primary-darkpink` (#610C39)
    *   **글자 (Text):** `--text-secondary-a` (#FFF4F6)
*   **Guest (CP)의 턴:**
    *   **배경 (Background):** `--primary-darkpurple` (#2E0645)
    *   **글자 (Text):** `--text-secondary-b` (#FDF9FF)

### 게임 종료 및 이탈 (Safe Exit)

#### 1. 종료 확인 팝업 (Exit Dialog)
게임 진행 중(Playing 상태) 뒤로가기 또는 종료 버튼을 누를 경우, 데이터 유실 방지를 위한 팝업을 표시합니다.

*   **Host (Member):** "게임을 저장하고 종료하시겠습니까?" (옵션: 저장 후 종료 / 그냥 종료 / 취소)
*   **Guest (Anonymous):** "로그인하지 않으면 게임 기록이 사라질 수 있습니다. 종료하시겠습니까?" (옵션: 종료 / 취소)
