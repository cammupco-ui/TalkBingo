# TalkBingo 수익화 전략 (Monetization Strategy)

> 최종 업데이트: 2026-02-10

---

## 1. 수익 모델 개요

| 구분 | 방식 | 역할 |
|------|------|------|
| **주 수입원** | 광고 시청 (보상형 + 전면 광고) | 전체 매출의 70~85% 목표 |
| **보조 수입원** | 포인트 구매 → 광고 제거 | 전체 매출의 15~30% 목표 |

## 2. 광고 유형별 상세

### 2-1. 보상형 광고 (Rewarded Video Ad)

| 항목 | 내용 |
|------|------|
| **광고 길이** | 15초 ~ 30초 (대부분 30초, 5초 후 스킵 불가) |
| **시청 조건** | 유저가 자발적으로 "광고 보기" 버튼 클릭 |
| **보상** | 광고 시청 완료 시 AP(Activity Point) 지급 |
| **eCPM (한국)** | **$20 ~ $40** (1,000회 노출당 수익) |
| **1회 시청 수익** | **약 $0.02 ~ $0.04** (한화 약 **27원 ~ 55원**) |
| **유저 경험** | 자발적 시청 → 거부감 낮음, 완료율 높음 |

> [!TIP]
> 보상형 광고는 모든 광고 유형 중 **eCPM이 가장 높고**, 유저 이탈률이 가장 낮습니다. 주 수입원으로 적합합니다.

#### 보상형 광고 노출 시점 (추천)
- 🎯 **게임 종료 후** — "광고 보고 보너스 AP 받기"
- 🎯 **AP 부족 시** — "광고 보고 AP 충전하기"
- 🎯 **힌트 사용 시** — "광고 보고 무료 힌트 받기"

### 2-2. 전면 광고 (Interstitial Ad) — ✅ 이미 구현됨

| 항목 | 내용 |
|------|------|
| **광고 길이** | 5초 ~ 15초 (5초 후 닫기 가능) |
| **시청 조건** | 자연스러운 전환 지점에서 자동 표시 |
| **eCPM (한국)** | **$10 ~ $20** |
| **1회 노출 수익** | **약 $0.01 ~ $0.02** (한화 약 **14원 ~ 27원**) |
| **유저 경험** | 강제 노출 → 적절한 빈도 조절 필요 |

#### 전면 광고 노출 시점
- 게임 종료/ 결과 화면 전환 시
- 3~4게임마다 1회 (과도한 노출 방지)

### 2-3. 배너 광고 (Banner Ad) — 선택 사항

| 항목 | 내용 |
|------|------|
| **eCPM (한국)** | **$1 ~ $5** |
| **1회 노출 수익** | **약 $0.001 ~ $0.005** |
| **적합 위치** | 홈 화면, 설정 화면 하단 |
| **권장** | ⚠️ 낮은 수익 대비 UX 저하. 초기에는 비추천 |

## 5. 광고 제거 포인트 가격 산정

### 산출 근거

유저 1명이 광고를 보면서 **발생시키는 평균 수익**을 기반으로, 광고 제거 결제가 그 수익을 **충분히 보상**해야 합니다.

#### 유저 1명의 월간 광고 수익

| 항목 | 계산 |
|------|------|
| 일일 보상형 광고 | 3회 × $0.03 = **$0.09** |
| 일일 전면 광고 | 1회 × $0.015 = **$0.015** |
| **일일 합계** | **$0.105** (약 ₩145) |
| **월간 합계** (30일) | **$3.15** (약 ₩4,350) |

## 8. 현재 포인트 시스템 분석 (GAME_RULES.md 기준)

### 8-1. 현재 포인트 구조

> 출처: [GAME_RULES.md](file:///Users/anmijung/Desktop/TalkBingo/doc/GAME_RULES.md)

| 포인트 | 약어 | 획득 방법 | 사용처 |
|--------|------|----------|--------|
| **Victory Point** | VP | 승리 시 **+50 VP** | 광고 제거 (**200 VP/게임**) |
| **Activity Point** | AP | 빙고줄: **+20/+40/+60**, 챌린지 공격: **+10**, 방어: **+5** | **100 AP → 50 VP** 전환 |
| **Experience Point** | EP | 셀(칸) 획득 시 **+1 EP** | **100 EP → 50 VP** 전환 |
| **Trust Score** | TS | 게임 종료 후 평가 (1~5점) | 평판 시스템 |

### 8-2. ⚠️ 현재 구조의 문제점

> [!WARNING]
> **현재 포인트 구조는 수익화와 충돌합니다.**

#### 문제 1: 광고 제거가 너무 저렴함
```
현재: 광고 제거 = 200 VP / 게임
승리 1회 = +50 VP → 4판 이기면 1게임 광고 제거 가능
AP 전환: 빙고 2줄(+40 AP) × 3게임 = 120 AP → 50 VP
EP 전환: 셀 25개 × 4게임 = 100 EP → 50 VP

→ 약 4~6게임이면 무료로 광고 제거 가능
→ 유저가 결제할 동기가 전혀 없음 ❌
```

#### 문제 2: VP가 이중 역할 수행
- VP는 **게임 성과 지표** (승리 보상) → 랭킹/명예용
- VP는 동시에 **화폐** (광고 제거 결제) → 경제용
- 하나의 포인트가 두 역할을 하면 **밸런스 조절이 불가능**

#### 문제 3: AP/EP → VP 전환이 VP 인플레이션 유발
- 활발한 유저일수록 VP가 자동으로 쌓임
- 광고 제거가 VP로 가능하므로, 열심히 플레이 = 영원히 무료
- **수익 = 0** 인 구조

## 9. 💡 포인트 구조 개편 제안

### 9-1. 핵심 원칙

```
🎮 게임 포인트 (GP)  = 게임에서 획득, 게임 기능에 사용 → 결제와 분리
💎 밸류 포인트 (VP)  = 현금으로 구매, 프리미엄 기능에 사용 → 수익화 전용
```

> [!IMPORTANT]
> **게임 활동 보상 ↔ 유료 결제를 완전히 분리**해야 수익화가 가능합니다.
> 무료 플레이로 유료 기능을 무한히 얻을 수 있으면 결제 동기가 사라집니다.

### 9-2. 개편안 — 이중 화폐 시스템

#### A. 게임 포인트 (GP) — 무료, 게임 내 획득

| 기존 | 개편 | 획득 방법 | 용도 |
|------|------|----------|------|
| VP (+50) | **GP +50** | 게임 승리 | 누적 통계 |
| AP (+20/40/60) | **GP +20/40/60** | 빙고 줄 완성 | 누적 통계 |
| AP (+10/+5) | **GP +10/+5** | 챌린지 공격/방어 성공 | 누적 통계 |
| EP (+1) | **GP +1** | 셀 획득 | 누적 통계 |
| 100 AP→50 VP | ❌ **삭제** | — | — |
| 100 EP→50 VP | ❌ **삭제** | — | — |

**GP 사용처 (결제와 무관, 현재 서비스 기준):**
- 📊 **프로필 누적 통계** — 총 GP, 승률, 게임 횟수 등 표시
- � **향후 확장 가능** — 랭킹, 칭호, 뱃지 등 (현재 미구현)

#### B. 밸류 포인트 (VP) — 유료, 현금 구매 또는 보상형 광고

| 획득 방법 | VP 지급 | 비고 |
|----------|---------|------|
| **IAP 구매** (₩1,100) | +100 VP | 현금 결제 |
| **IAP 구매** (₩5,500) | +600 VP | 보너스 +50 VP |
| **IAP 구매** (₩11,000) | +1,300 VP | **가성비 추천** ⭐ 보너스 +100 VP |
| **IAP 구매** (₩33,000) | +4,000 VP | 보너스 +500 VP |
| **IAP 구매** (₩64,000) | +8,000 VP | 보너스 +1,000 VP |
| **보상형 광고 시청** (30초) | +5 VP | 유저당 일일 최대 10회 (50 VP/일) |

**VP 사용처 (현재 서비스 기준):**
- 🚫 **1게임 광고 제거** — **200 VP**
- 🚫 **영구 광고 제거** — **8,000 VP** (VP 누적 구매 또는 일괄 구매)
- 🎁 **향후 확장 가능** — 특별 이모지, 프리미엄 질문 팩 등 (현재 미구현)

### 9-3. 개편 전후 비교

#### 비교: "무과금 유저가 광고 제거하려면?"

| | 현재 (구 VP) | 개편 후 (VP) |
|---|-----------|-------------|
| **필요 포인트** | 200 VP | 200 VP |
| **게임플레이로 획득** | 4판 승리 (200 VP) | ❌ 불가능 (GP ≠ VP) |
| **보상형 광고로 획득** | 없음 | 40회 × 5 VP = 200 VP (**4일**) |
| **현금 결제로** | 불가 | ₩1,100 (100 VP) × 2 = **₩2,200** |
| **결제 동기** | ❌ 없음 | ✅ 있음 |

#### 비교: "열심히 플레이하는 유저의 경험"

| | 현재 | 개편 후 |
|---|------|--------|
| **게임 보상** | VP 쌓임 → 광고 제거에 소비 | GP 쌓임 → 프로필 누적 통계 성장 |
| **성취감** | VP 소비하면 사라짐 😢 | GP는 영구 누적, 통계에 반영 🎉 |
| **광고 관련** | 무료로 계속 제거 가능 | 보상형 광고 시청으로 VP 모으기 |
| **결제 유도** | ❌ 전무 | ✅ "VP 조금만 더 있으면 광고 제거!" |

### 9-4. 보상형 광고 ↔ VP 밸런스

> [!TIP]
> **핵심 공식: 보상형 광고 1회(30초) = 5 VP**
> 
> - 1게임 광고 제거(200 VP) = 보상형 광고 **40회** = 약 **20분** 시청
> - 이것이 유저에게 "광고 좀 볼까" vs "₩2,200 결제할까" 선택지를 자연스럽게 제공
> - 일일 상한 10회(50 VP) → 4일간 꾸준히 광고 봐야 1게임 제거 가능 → **결제 유도**
